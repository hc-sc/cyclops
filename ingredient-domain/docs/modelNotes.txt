Model Notes
===========
Unfortunately, the Ingredient Database model has not been kept up-to-date.

Recent refactoring of the Monograph portion of that model was required and that
has been saved separately as its own model file. Even though the entire
Ingredient Database model is in that file, only the Monograph related objects
themselves have been updated in it. Note that there are Classes in this project
that are not considered part of the domain model - they are implementation
artifacts. As a result, they are not in the model's class diagrams.
